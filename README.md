# 🔐 Password Security Analyzer

A Flask-based web application that checks password strength using security rules and entropy calculation.

## 🚀 Features

- Password Strength (Weak, Medium, Strong)
- Entropy Calculation
- Uppercase, Lowercase, Number & Special Character Detection
- Live Strength Meter
- Responsive UI

## 🛠️ Tech Stack

- Python
- Flask
- HTML
- CSS
- JavaScript

## ▶️ Run

```bash
git clone https://github.com/Khushi7007/Decode-Lab-1-project-password-strength-checker.git
cd Decode-Lab-1-project-password-strength-checker
pip install flask
python app.py
```

Open:
```
http://127.0.0.1:5002
```
